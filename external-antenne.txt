У Samsung S21 Ultra действительно только один USB-C порт, но есть несколько эффективных решений для подключения внешних антенн без потери USB управления.
Решение через USB-C Hub
USB-C Hub с поддержкой OTG:
Samsung S21 Ultra
    ↓ USB-C
USB-C Hub (с OTG + Power Delivery)
    ├── USB-A → Arduino (управление)
    ├── USB-C → Зарядка
    └── Остальные порты свободны
Рекомендуемые модели:

Anker PowerExpand+ 7-in-1 — надежный, с OTG
UGREEN 9-in-1 USB-C Hub — бюджетный вариант
HyperDrive 6-in-1 — компактный для мобильных устройств
